# 更新日志

## V2.4 — 2026-05-29

### ✨ 新增功能

- **截图选点设置点击位置**：点击偏移设置改为在截图上直接选择坐标。点击"在截图上选择"按钮 → 截取当前屏幕 → 用户在截图上单击选择目标位置 → 执行时直接点击该绝对坐标，不再经过模板匹配 + 偏移量计算。支持"清除坐标"恢复为默认的模板匹配中心点击。
- **截图窗口自适应尺寸**：框选识别区域和截图选点两个窗口默认大小改为屏幕分辨率的 85%，并支持拖拽缩放，截图的显示尺寸大幅提升。

### 🔧 优化

- **按钮文字调整**：主窗口"选择目标窗口"改为"绑定目标窗口"，语义更清晰。
- **版本号更新**：程序标题和打包脚本版本号从 v2.0 升级为 v2.4。

### 🔄 数据兼容性

- 保留旧的 `click_offset_x/click_offset_y` 字段（偏移量模式），新增 `click_x/click_y` 字段（绝对坐标模式）。
- 执行引擎优先使用绝对坐标；若未设置，自动回退到模板匹配中心 + 偏移量逻辑。
- 现有 `config.json` 配置不受影响，可正常加载。
